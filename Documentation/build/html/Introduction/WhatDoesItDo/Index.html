What does it do?¶

Implements one or more instances of the Trumbowyg Editor along with specific IDs for each instance (click here to see a demo).

Those IDs are used to access the editor to perform further processment of it’s content.

The editor can be customized providing a proper JSON as described in the Trumbowyg Documentation.

Typowyg also includes a plugin that enables the user to embed or link content of his own google drive.

After clicking the Google Drive icon and logging in a browser will popup that enables the user to create and rename directories or upload files prior to select the one to either embed or link to.

When embedded google’s viewer are used to display the type of file. Those viewer are for example able to zoom into images, navigate PDFs and browse ZIP-Files.

Simply linking text to Google Drive content is, of course, also possible.
