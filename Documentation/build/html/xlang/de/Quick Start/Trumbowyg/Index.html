Trumbowyg Editor¶

  • Das Viewer-Typoscript zur Site - Template hinzufügen.

  • Den Typowyg - Viewer zum Seiteninhalt hinzufügen.

  • Die Konfiguration des Elements öffnen, zur Typo WYG Palette wechseln und eine Instanz-ID eintragen.

  • Eine entsprechende Konfiguration eingeben.

  • Die zu ladenden Plugin-Bibliotheken (wenn z.B. speechrecognition in der Konfiguration genutzt wird, ist es auch notwendig die Speechrecognition - Bibliothek hier hinzuzufügen damit diese Funktionalität zur Verfügung steht und der Mikrofon - Button im Editor angezeigt wird ) einbinden.

  • Javascript nutzen um auf den vom Nutzer in den Editor eingegeben Inhalt zuzugreifen.
