Questa estensione implementa l'editor `Trumbowyg Editor <https://alex-d.github.io/Trumbowyg/>`_.
Con una dimensione di 20kb, questo editor è leggero rispetto ai soliti editor con una dimensione di 40kb.
Tuttavia, l'editor offre molte funzioni che possono essere facilmente integrate tramite plugin, come la funzione di dettatura.

Typowyg include anche un plugin che consente agli utenti di navigare nel proprio `Google Drive <https://drive.google.com>`_, caricare file e incorporarli o collegarli al contenuto testuale dell'editor.
