Diese Extension implementiert den `Trumbowyg Editor <https://alex-d.github.io/Trumbowyg/>`_.
Dieser Editor ist mit 20kb ein Leichtgewicht gegenüber den sonst üblichen Editoren mit einer Größe von 40kb.
Trotzdem bietet der Editor viele Features die über Plugins leicht eingebunden werden können wie z.B. die Diktierfunktion.

Typowyg enthält zusätzlich ein Plugin welches es dem Nutzer erlaubt in seinem eigenen `Google Drive <https://drive.google.com>`_ zu navigieren, dort Dateien hochzuladen und diese in den Textinhalt des Editors entweder einzubinden oder zu verlinken.
